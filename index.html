Hola, soy Bautista Hidalgo, y estoy aprendiendo a hacer sitios web

Estoy aprendiendo desarrollo web, porque quiero lograr dos objetivos, el primero es, que la universiad, una vez que salga de la secundaria, se me haga mas fácil, el segundo objetivo es trabajar en las vacaciones o en mis tiempos libres para poder ganar plata y comprarme cosas que yo quiero, salir a comer con mis amigos, etc.

Que voy a estar aprendiendo?

En el transcurso de este curso voy a aprender sobre tres programas que son: HTML, CSS y JavaScript (JS), estos tres programas son fundamentales a la hora de hacer una pagina web y me ayudaran a entender mejor el funcionamiento que hay destras de cada sitio web.
